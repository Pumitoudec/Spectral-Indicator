# Spectral Inficator  
### UAV-Based Multispectral Analysis System with Spectral Index Calculator

---

## Description (English)

A Python-based software for visualization, editing, calibration, and processing of multispectral GeoTIFF images.  
The system allows the computation of up to **40 spectral indices** using the **Green, Red, Red Edge, and Near-Infrared (NIR)** bands.  

It also provides an intuitive and organized workflow for extracting average information from user-defined **Regions of Interest (ROIs)**, which can be easily and manually created.

---

## Descripción (Español)

Programa desarrollado en Python para la visualización, edición, calibración y procesamiento de imágenes multiespectrales en formato **GeoTIFF**.  
El sistema permite calcular hasta **40 índices espectrales** a partir de las bandas **Green, Red, Red Edge y Near-Infrared (NIR)**.  

Además, permite de manera simple y ordenada la extracción de información promedio mediante **Regiones de Interés (ROIs)** definidas por el usuario, las cuales pueden ser creadas de forma manual y sencilla.

---

## Main Features

- Multispectral GeoTIFF visualization and processing  
- Radiometric calibration  
- Computation of 40 spectral indices  
- ROI-based statistical extraction  
- UAV / Drone oriented workflow  
- Fully developed in Python
